MongoDB 是一種開源的、高性能的 NoSQL（非關聯型資料庫）資料庫系統，它以文件形式儲存資料，適合處理大量非結構化或半結構化的資料。MongoDB 使用 BSON 格式的文件來儲存資料，與 JSON 格式幾乎一樣，這使得資料存取相當靈活，並且非常適合在 Web 開發中應用。
