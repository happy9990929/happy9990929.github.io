Node.js 是一個開源且跨平台的運行時環境，讓我們能夠在伺服器端運行 JavaScript 應用程式。
這讓前端工程師不僅能在瀏覽器中運行 JavaScript，還可以在伺服器上運行相同的語言，實現全端開發的夢想。
我們將介紹如何在 Node.js 環境中執行 JavaScript 檔案，之後會再深入了解 node 命令以及常見的檔案路徑操作。

結語

後端開發跟前端明顯不一樣的地方是，前端可以看到畫面，但後端結果會是在 Terminal 中顯示，
在後端我們會常常使用 Terminal、postman 來操作，
不過因為 Node.js 是在 JavaScript 環境中運行，所以前端工程師也可以很容易上手！
