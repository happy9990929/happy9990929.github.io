瀏覽器視窗alert、comfirm、prompt

發表於 2023-02-09 分類於 Javascript 閱讀次數： Disqus：

相信大家都有用過 alert 視窗，除了 alert 其實還有其他的視窗可以使用，
JavaScript 提供三個方法 alert(), confirm(), prompt() 讓你可以使用瀏覽器內建的對話視窗。



alert()

alert() 跳出提示 (警告) 對話視窗



confirm()

confirm() 跳出需要使用者確認的對話視窗，對話視窗中會有確定及取消兩個按鈕



prompt()

prompt() 跳出需輸入文字的對話視窗

  • 第一個參數 message 是給使用者看得提示文字
  • 第二個參數 default 是輸入框中的預設文字
  • prompt() 執行後會返回使用者輸入的字串 (或得到空字串)，如果使用者按取消按鈕則會返回 null
