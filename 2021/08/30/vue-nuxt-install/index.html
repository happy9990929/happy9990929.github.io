安裝 SASS

Nuxt 預設是沒有辦法使用 SASS 的，如需要可以 npm 安裝 SASS

components: false

這個功能是 nuxt v2.13 時候所導入的功能，可以讓你不需要 import 就可以載入組件，但是這會造成不知道組件的正確路徑以及可能會有命名的相關問題，所以不推薦使用
在 nuxt.config.js 裡面把 components 這個欄位設定成 false，或是砍掉這個設定
