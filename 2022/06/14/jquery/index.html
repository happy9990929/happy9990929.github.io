jQuery常用方法大全

發表於 2022-06-14 更新於 2022-06-27 分類於 jQuery 閱讀次數： Disqus：


jQuery 是一個 Javascript 的函示庫，透過它可以簡單的撰寫做出各種網頁效果。
以下是 jQuery 一些常用的方法，方便查閱記錄。

元素查找

prev()

取得同層上一個元素


next()

取得同層下一個元素


nextAll()

取得元素之後的所有兄弟元素


siblings()

取得所有同级元素


children()

選取元素的子元素


parent()

選取元素的父元素中符合選擇器的元素


parents()

選取元素的祖先元素中符合選擇器的元素


parentsUntil()

選取元素的祖先元素中直到遇到第一個符合選擇器的元素(不含此元素)


before()

在DOM之前插入元素


after()

在DOM後插入元素


prepend()

將元素添加到被選元素的開頭


append()

將元素添加到被選元素的結尾


remove()

移除元素


closest()

尋找離當前元素最近的祖先元素


find()

查找元素底下的子元素
與 $('p .text') 相同


filter()

查找元素同層
與 $('p.text') 相同


has()

選取元素中子孫元素符合選擇器的元素


eq()

元素中符合索引值的元素


屬性與 CSS

attr()

設定元素的屬性和值


css()

設定CSS樣式


animate()

控制CSS動畫效果


stop()

元素動畫完成前執行停止
true 可中斷執行中動畫
false 不可中斷執行中動畫


addClass()

加入class


removeClass()

移除class


toggleClass()

收合class，如果有class就移除，沒有就加入


操作

ready()

DOM都載入後執行

也可為


clone()

生成被選元素的副本，包含子節點、文本和屬性


each()

對每個選取元素進行處理


create element

使用標籤建立元素


on()

執行動作
mouseover、mouseout、mousedown、mouseup、mousemove、click、dbclick、keydown、keyup、focus、blur、change、resize、scroll
